# Chlorophylls
<p>Utility for determining the maintenance of chlorophylls A & B in plant material</p>
<p>Run the utility with specified keys</p>
<p><b>chloro -d [mass g] [filtrate ml] [photoprobe ml] [solvent ml] [D 665] [D 649]</b></p>
